# Περιεχόμενο του φακέλου
Τα Pinns είναι νευρωνικά δίκτυα (δηλαδή, λίγο πολύ, "τεχνητή νοημοσύνη") που στοχεύουν να επιλύσουν φυσικά προβλήματα. Σε αυτό το φάκελο βρίσκονται (κατά το δυνατόν απλές) εφαρμογές σε διαδικασίες που συχνά εμφανίζονται σε προβλήματα Φυσικής, όπως η επίλυση Διαφορικών Εξισώσεων.

Κατά την πιο πρόσφατη ενημέρωση του παρόντος (10/2/2026) ο φάκελος περιλαμβάνει:
* Έναν κώδικα σε python για την επίλυση μίας απλής γραμμικής ΣΔΕ 1ης τάξης. Με απλές τροποποιήσεις, ο κώδικας μπορεί να χρησιμοποιηθεί και για την επίλυση κάθε άλλης πρωτοτάξιας ΣΔΕ. Με λιγότερο απλές τροποποιήσεις, μπορεί να αποτελέσει τη βάση για την επίλυση ΣΔΕ ανώτερης τάξης/ συστημάτων ΣΔΕ.
